# Device Management Application

A simple web-based device management system to track and manage devices.

## Features

- Add new devices with name, type, and status
- View all devices in a table format
- Delete devices
- Data persistence using localStorage
- Responsive design

## How to Use

1. Open `index.html` in your web browser
2. Fill in the device information (Name, Type, Status)
3. Click "Add Device" to add the device to the list
4. Use the "Delete" button to remove devices
5. Data is automatically saved to your browser's local storage

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and layout
- `script.js` - JavaScript functionality
- `README.md` - This file

## Getting Started

Simply double-click on `index.html` or open it in any modern web browser to start using the application.
